# ユーザー テーマ

Themeプロパティはテーマの編集を可能にし、ユーザーが異なる背景画像、色設定、アイコン、フォントを使用して、アプリケーションの側面をカスタマイズすることができます。

**ホームページ > デフォルトのテーマ**

![ホームページ、デフォルトのテーマ](../../../images/user-documentation/content-user-manual/customize/67_Homepage_Default_theme.png)
