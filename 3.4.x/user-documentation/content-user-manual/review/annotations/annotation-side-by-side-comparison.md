# サイドバイサイド比較

注釈ページには、アセットの異なるバージョンを比較するための横並び比較ツールが用意されています。ユーザーは、アノテーション ページの [比較] ボタンをクリックすると、並べて比較ツールのページに移動できます。

アノテーションの比較ボタンをクリックすると、横並び比較ツールのページに移動します。
