      バージョンの確認方法

      利用している製品のバージョン確認の手順は以下の通りです:
      1. 利用している Sitecore Content Hub のサーバにアクセスします (e.g. my-instance.com/en-us).
      2. URL から /en-us を削除して、/api/status を入力して Enter で確定させます.
      3. JSON データの中に, "product_version" プロパティが含まれます。
      4. '+' の前がバージョン番号になっています。この例では、値は 3.3.0 です。
